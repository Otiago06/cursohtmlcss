### Representação de Dados
MB -> Megabytes (armazenamento)
Mb - Megabit (transmição de dados)
- 2¹º (2 elevado a 10)
- 1024 bytes = 1KB
- 1024 KB = 1 MB
- 1024 MB = 1 GB
- 1024 GB = 1 TB
- 1024 TB = 1 PB
- 1024 PB = 1 EB
- 1024 EB = 1 ZB
- 1024 ZB = 1 YB
